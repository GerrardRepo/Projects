# AI Chatbot

A simple command line chatbot built with **LangChain** and **OpenAI** that streams responses by token in the terminal.

## Features

- Chat with an OpenAI model of your choosing from the terminal
- Error handling included
- Utilisation of API for a simpler code

---

## Tech Stack

- Python
- LangChain
- LangChain OpenAI
- python-dotenv
