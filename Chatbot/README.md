# AI Chatbot

A simple command-line chatbot built with **LangChain** and **OpenAI** that streams responses token-by-token in the terminal.

## Features

- ✅ Chat with an OpenAI model from the terminal
- ✅ Streaming output (response appears live as it is generated)
- ✅ Uses `.env` for API key security
- ✅ Basic error handling
- ✅ Clean and beginner-friendly code structure

---

## Tech Stack

- Python
- LangChain
- LangChain OpenAI
- python-dotenv
